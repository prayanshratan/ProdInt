## Common Questions

**Q: Do I need an API key?**  
A: No! We provide a default key. But your own key is better.

**Q: What file formats are supported?**  
A: Upload: DOCX, TXT, MD | Download: MD, DOCX

**Q: Can I customize templates?**  
A: Yes! Upload any template format you want.

**Q: Is my data secure?**  
A: Yes! Passwords are hashed, sessions are secure, API keys are encrypted.

**Q: Can I use it offline?**  
A: No, AI generation requires internet connection.
